# demo url-shortner
Demo available on [AWS](http://18.202.239.19)

## Setup Instructions

### With Docker

```
git clone https://github.com/JackRawlinson/url-shortner.git
cd url-shortner

composer install
docker-compose build
docker-compose up
```

Navigate to 127.0.0.1:8090
